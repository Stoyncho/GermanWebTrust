## 🎯 Как да стартирате

### Метод 1: Директно отваряне
1. Разархивирайте файловете
2. Отворете `index.html` в браузър
3. Готово! 🎉

### Метод 2: С Live Server (препоръчително)
```bash
# Инсталирайте Live Server чрез VS Code
# Или използвайте Python:
python -m http.server 8000

# Отворете http://localhost:8000
```

### Метод 3: За качване онлайн
1. Качете всички файлове на вашия хостинг
2. Уверете се, че пътищата са правилни
3. Тествайте на различни устройства

## 🛠️ Настройки

### Промяна на цветове
Редактирайте CSS променливите в `:root` в `css/style.css`:

```css
:root {
    --primary: #ВАШИЯТ_ЦВЯТ;
    --secondary: #ВАШИЯТ_ЦВЯТ;
}
```

### Промяна на текстове
Редактирайте директно в `index.html`. Всички текстове са на немски за целевия пазар.

### Добавяне на изображения
1. Сложете изображения в `images/` папката
2. Обновете пътищата в HTML
3. Препоръчително: WebP формат за по-добра производителност
